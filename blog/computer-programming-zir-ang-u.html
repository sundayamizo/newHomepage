Computer Programmig Zir Ang U!
24/01/2016 02:17 PM, Ramhlun South, Aizawl

Enge  Computer Programming (emaw Programming) kan tih fo hi? Enge code kan tih thin hi? ahn sawifiah kak dawn chuan Computer chhungril (computer architecture) lam kan sawi thui viau a ngai dawn a, ka hun leh ka thiamna ten a daih dawn bik loh avangin kan lut thuk rih lo mai aniang chu!

A awlsam thei ang ber leh mi nawlpui in a kan hriat thiam theih ang zel turin han sawifiah ta ila; a hmasa in enge CODE hi?

Computer in a hriatthiam chhun chu 0 leh 1 chauh hi a ni, chu chu Binary language (machine langauge) an ti. TV ah emaw , khawi lai lai ah emaw a hnuai a mi ang khu i hmu tawh em ?


Binary codes

Heng number 0 leh 1 awm khawm thup hi number mai mai an ni hauh lo a, computer biak nan a kan tawng neih chhun an ni tlat! Mi thil ti tur e i hrilh dawn chuan i tawng emaw i zai zir kha a hriat thiam a ngai a ni tiraw? Chutiang chuan computer pawh hian thil ti thei viau mahse tih tur kan pek loh chuan a mah hian tih ngaihna a hre lo. Tihtur (instruction) kan pek hnuah chauh a tih tur te tha takin a ti thei thin. Chu tihtur kan hrilh theihna tawng chu he "binary language" hmang chauh hian a ni! A hnuaia kan sawi leh tur hi awm ta lo se han ngaituah teh, computer a hla ngaihthla tur ringawt pawn 0 leh 1 maktaduai chuang chhut hmasak phawt ngai se, a hla play lai pawn 0 leh 1 tho chhut char ngai se!!!!! Ngawi teh, mihring kut chet ran theih zawng hian a pha dawn lo hrim hrim a ni e!

Khawvel changkang zel hian awlsam zawk a thil tih dan tur a zawng zel a! Chu machine language chu mihring hriatthiam awlsam tur leh automatic a tih zung zung theih turin Programming language dang an siam a, chu chu tun a Computer Programming kan tih a kan zir ang hi (high-level language) te hi an lo ni ta a ni.
Mahse kan ti leh teh ang! Machine language leh High-level language inbiak pawh theih nan palai a awm ve leh a , chu chu Assembly language kan tih te hi an ni! A hnuai a lem ah khuan kan chiang mai awm e!

Computer hnena tih tur hrilh dan kawng


A chung a kan hmuh ah khian C, Pascal leh C++ kan hmuh te khi High-level language kan tih, Computer Programming course hrang hranga kan zir thin te zinga mi an ni. Mihring ten kan hriatthiam theih dan zawng a Computer tih tur kan hrilhna tawng an ni. Chutia kan tih tur hrilh chu Compiler/IDE hmangin Assembly language-ah kan let a, Assembly language chu Assembler in Machine language (binary) ah a lo let ve leh a, chu binary language Hardware (Laptop, PC,TV, Phone,etc.,) te hian an lo hrethiam ta a, kan tih tur sawi ang ang chu a nih ang thlap in min tih hlawtlinsak ta thin!

Heng ang pui pui kan siam thei lo a nih pawn a hnathawh  dan te, enge a awm zia tal hi hrethiam i la, kan Technology khawvel thlir dan hi a danglam ngei a rinawm! Heng thil zir tur hian thil tam angai miah lo a, i hma a Computer a awm a, a zirna tur source Internet emaw lehkhabu emaw a awm chuan i zir thei vek a ni tih hria ang che! Internet ah Ebook a thlawn tam tak a awm a, YouTube an Video Tutorial en sen loh, thiam sen loh a awm bawk! Heng te hi i zir peih chuan  a mak a mak a tam a awm a ni!



Tikhan kan hriatthiam ve ruai ruai tan chu i han beisei ila! Kan sawi fo thin a, Computer Age a ni tawh tih hi!  Hun a kal zel ang a, khawvel changkang zel hian hna tha thawk thei tur chuan heng High-level language te hi a la phut lo ang tih a sawi theih loh! Tunlaiah pawh hna kan zawng a, Computer hman dan tlangpui kan hriat loh chuan iai-awm kan ni nghal zel ta mai! Heng hna kan zawn a Computer thiam tur a min duh ang level tam zawk hi chu Programming kan tih nen chuan a la inhlat viau awm e. Computer basic tak tak chauh hre tur in min beisei tlangpui thin. Heng pawh kan tih theih loh chuan khawvel kal rual a kan pen thiam ve theih loh hun hi a la thleng thei mai awm e!

Ama'rawhchu, in demna ngawt chi erawh a ni chuang hauh lo! Lirthei pawh khalh thiam tur chuan a lirthei a kha a awm hmasak phawt a ngai! Computer a awm miau loh chuan computer a thiam tak tak theih loh a, khawih tam ngai tak thil ve tho a ni! Computer thiam hi a la pawimawh dawn chauh tih hria ila, Programming language harsa pui pui kan thiam lo a nih paw'n a basic tal hi chu i han zir tlang teh ang u!
